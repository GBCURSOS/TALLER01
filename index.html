FSB1_152 TALLER 01

Taller sobre HTML y CSS & GIT

La disciplina, tarde ó temprano, vencerá a la inteligencia

CONDICIONES DE ENTREGA

  1. Cada ejercicio debe estar resuelto en un archivo .html independiente y el .css correspondiente debe estar en un archivo aparte.
  2. Debe quedar en un repositorio con el nombre F152T01 y desplegar la página web correspondiente
  3. De este taller se creara una página, por tanto debe tener una sección de footer, en el cual se indique el nombre del estudiante y su identificación correspondiente
  4. La página index.html debe estar conformada por un índice, con enlaces a cada uno de los archivos .html correspondientes.
  5. Debe mandar el enlace de la página por el whatsapp del grupo.

EJERCICIOS

  1. Ejercicio 00

    Cree la estructura de carpetas correspondiente donde se guardaran los archivos solución a los diferentes ejercicios.

    Cree una carpeta nueva para éste taller. En este caso se llama FSB15201:

    Cree la siguiente estructura de carpetas:

    1. En la carpeta CSS, deben quedar los archivos .css que se van a crear.
    2. En la carpeta HTML, deben quedar los archivos .html que se van a crear.
    3. En la carpeta FOTOS, deben quedar las fotos que se utilicen.

  2. Ejercicio 01

    Cree una página web con tres párrafos, cada uno de 150 palabras, con las siguientes condiciones:

    1. Cada párrafo debe tener un id con la identificación: uno, dos y tres, respectivamente.
    2. Cada párrafo debe tener un color diferente para el texto.

  3. Ejercicio 02

    Cree una página web con tres fotos con título para cada una de las fotos. Las fotos las puede descargar de undraw.co. Recuerde que las fotos deben ir en la carpeta RECURSOS > FOTOS.

  4. Ejercicio 03

    Cree una página con la siguiente estructura:

    
                        <header>
                            <h1>Encabezado h1 en el Header</h1>
                        </header>
                        <main>
                            <h1>Encabezado h1 en el main</h1>
                            <section>
                                <h2>Encabezado en la sección Uno</h2>
                                <div>
                                <p>División en la sección Uno</p>
                                </div>
                            </section>
                            <section>
                                <h2>Encabezado en la sección Dos</h2>
                                <div>
                                <p>División en la sección Dos</p>
                                </div>
                            </section>
                            <section>
                                <h2>Encabezado en la sección Tres</h2>
                                <div>
                                <p>División en la sección TRes</p>
                                </div>
                            </section>
                        </main>
    1. Cada elemento header, main, section y div, deben tener un color de fondo diferente. El mismo color para el correspondiente elemento.
    2. Los encabezados deben tener el caracter de color rojo.

  5. Ejercicio 04

    Realice una página con una tabla de verduras, donde las columnas sean nombre, color y beneficio. Deben haber como mínimo diez ítems

Atajo Descripción
etiqueta.nombre_de_la_class + Tab Crear un elemento con clase (Emmet)
etiqueta#nombre_del_id + Tab Crear un elemento con id (Emmet)
etiqueta.nombre_de_la_class#nombre_del_id + Tab Crear un elemento con class e id (Emmet)
etiqueta.nombre_de_la_class.nombre_de_la_class + Tab Crear un elemento con dos class (Emmet)
Ctrl + Shift + K Borrar una línea
Alt + ↑ / ↓ Mover una línea
Shift + Alt + ↑ / ↓ Copiar una línea
Shift + Alt + F Organizar el código (formatear la indentación)
Ctrl + A Seleccionar todo el código
Ctrl + K + Cttrl C Comentar ó descomentar una línea
